# CalDAV 同步

将思源笔记属性表（Attribute View）与 CalDAV 日历双向同步，内置可视化日历面板与任务管理。

## 功能

- **双向同步** — 思源 ↔ CalDAV 增删改自动对齐，Last-Write-Wins 冲突策略
- **多账户** — 同时接入多个 CalDAV 日历（个人 / 工作 / 家庭）
- **日历面板** — 月 / 日 / 年三视图，鼠标滚轮切换月份，折叠/展开自适应布局
- **任务管理** — 优先级 · 进度 · 勾选框三维度管理，进度联动日期自动填充
- **自动推进** — 到达开始/结束时间自动升级进度（只升不降）
- **面板内编辑** — 新建/编辑事件，无需切换属性表
- **智能修复** — UID 不匹配的孤儿事件自动关联
- **熔断保护** — 连续错误自动暂停 + 思源数据为空时拒绝同步
- **暗色适配** — 完整支持思源亮色/暗色主题

## 快速开始

### 1. 创建属性表

在思源中新建一个属性表（Attribute View）。

### 2. 创建列

按下表创建 11 列，并记录每列的 **列 ID**（列头 → 列设置 → 查看）：

| 列名 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 标题 | text | ✅ | 事件标题 |
| 开始时间 | date | ✅ | 事件开始 |
| 结束时间 | date | | 事件结束 |
| UID | text | ✅ | 同步自动填充 |
| 同步时间 | date | ✅ | 同步自动更新 |
| 描述 | text | | 事件描述 |
| 提醒 | select | | 建议选项：无提醒、准点提醒、提前5分钟、提前15分钟、提前1小时、提前1天 |
| 账户 | select | ✅ | 同步自动填充 |
| 优先级 | select | | 建议选项：不重要、一般、重要、非常重要 |
| 进度 | select | | 建议选项：未开始、进行中、已完成、未完成 |
| ☑ | checkbox | | 勾选框 |

### 3. 配置插件

1. 点击顶栏日历图标 → 设置
2. 填写 **数据库 ID**（属性表 URL 中的 ID）
3. 填写各 **列 ID 映射**
4. 可选填写 **视图 ID**（锁定视图，防误删）

### 4. 添加 CalDAV 账户

1. 设置面板 → CalDAV 账户 → **+ 添加账户**
2. 填写：账户名称、服务器 URL、用户名、密码
3. 点击 **测试连接** 确认可用

### 5. 保存并同步

点击 **保存配置**，自动启动定时同步。

## 日历面板

点击顶栏日历图标打开。

| 视图 | 说明 |
|------|------|
| **月视图** | 当前月默认折叠（显示一周行），其他月默认展开；折叠用彩色圆点指示事件，展开用事件条 |
| **日视图** | 全天事件独立卡片，每小时一行，当前小时高亮 |
| **年视图** | 3×4 月卡片网格，每卡含迷你日历和事件数 |

**交互**：鼠标滚轮 / 左右方向键 / 触摸滑动切换月份；点击标题层级上钻（日→月→年）；"今天"按钮快速回到当前日期；ESC 关闭。

**侧栏**：显示选中日期的事件列表 + 待安排区（无日期事件按优先级排序）。点击 **+** 新建事件，点击事件标题编辑。

## 任务管理

### 优先级颜色

| 优先级 | 颜色 |
|--------|------|
| 非常重要 | 🔴 红色 |
| 重要 | 🟠 橙色 |
| 一般 | 🔵 蓝色（默认） |
| 不重要 | ⚪ 灰色 |

### 进度联动规则

| 规则 | 触发 | 自动动作 |
|------|------|---------|
| R1 | 进度 → 进行中 且 开始日期为空 | 填充开始日期 |
| R2 | 进度 → 已完成/未完成 且 结束日期为空 | 填充结束日期 |
| R3 | 进度 → 未开始 | 清空开始和结束日期 |

> 日期非空时不自动修改，尊重手动填写。

### 自动推进

同步时检测时间到达，自动升级进度（只升不降）：

- 到达开始时间 → 未开始 → 进行中
- 到达结束时间 → 进行中 → 已完成

### 勾选框

勾选框是进度的快捷映射：勾选 → 已完成 + 填充结束日期；取消 → 进行中。

## 同步引擎

三阶段策略：

| 阶段 | 操作 |
|------|------|
| 阶段零 | 智能修复 UID — 标题+时间模糊匹配孤儿事件 |
| 阶段一 | 删除检测 — 对比缓存，双向同步删除 |
| 阶段二&三 | 变动扫描 — 哈希对比，Last-Write-Wins 覆盖 |

**熔断保护**：连续错误达阈值（默认 5 次）自动暂停；思源数据为空时拒绝同步（安全熔断）。

**提醒同步**：支持无提醒、准点、提前5/15/60分钟、提前1天等选项的双向同步。

## CalDAV 服务器地址

URL 应指向日历端点（非账户根路径）：

| 服务器 | 示例 |
|--------|------|
| Radicale | `https://caldav.example.com/dav/user/calendar/` |
| Nextcloud | `https://cloud.example.com/remote.php/dav/calendars/user/personal/` |
| Baikal | `https://dav.example.com/dav/calendars/user/default/` |

> URL 末尾一般需要 `/`。

## 常见问题

**同步不生效？** 检查数据库 ID、列 ID 映射、CalDAV 账户连接，以及事件是否有开始时间。

**事件被误删？** 确认已填写视图 ID；安全熔断会自动中止同步。

**UID 列为空？** 首次同步自动生成并回写。

**密码安全？** 明文存储（建议使用应用专用密码，如 Nextcloud App Password）。

**重置同步？** 设置面板 → 重置缓存 → 下次同步重新全量比对。
